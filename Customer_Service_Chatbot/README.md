# AI Customer Service Chatbot

## Project Description

This project is an NLP-based Customer Service Chatbot developed using Python and Scikit-Learn.

The chatbot helps users by answering common customer support questions such as:

* Working Hours
* Contact Information
* Refund Policy
* Order Tracking
* Delivery Information

## Technologies Used

* Python
* NLP
* Scikit-Learn
* TF-IDF Vectorizer
* Cosine Similarity
* Tkinter GUI

## Features

* GUI-based chatbot
* Instant responses
* NLP-powered question matching
* Easy to customize

## How to Run

pip install -r requirements.txt

python gui_chatbot.py
